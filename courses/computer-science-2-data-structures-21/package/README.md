# computer-science-2-data-structures-21

## What This Package Is

This is a Correcta course package for a 21-day Computer Science 2 course. The package is Git-native and AI-gradable. It contains only course template content: manifests, grading configuration, syllabus, knowledge notes, and assignment definitions.

## Course Topic

Computer Science 2 with a focus on recursion, object-oriented design, linked data structures, stacks, queues, trees, traversal, search, and algorithm analysis.

## Duration

21 days.

## What Students Will Produce

Students will produce recursive functions, object-oriented modules, custom linked structures, tree traversal code, an algorithm analysis report, and a final data-structure-based Python application with tests and written documentation.

## Files Inside This Package

- correcta.course.json: main course manifest.
- grading.json: course grading scale and assignment weights.
- syllabus.md: course overview, schedule, assignments, and expectations.
- README.md: package description and import notes.
- knowledge/: concise instructional notes for each module.
- assignments/: self-contained assignment definitions with rubrics.

## Import Notes

Import this package as a `.corr` archive. Correcta should install it to `courses/computer-science-2-data-structures-21` according to the manifest. Runtime learner data should be created by Correcta after import and is not included in this package.
