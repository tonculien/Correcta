# 14-Day Software Engineering Foundations

This Correcta course is a compact project-based simulation of core software engineering work.

## Course Goal

By the end of the course, the student should be able to move from a vague software idea to a small but documented and testable software project.

## What This Course Emphasizes

- Requirements before implementation
- Clear scope control
- Basic design documentation
- Git workflow and commit discipline
- Testing and edge cases
- Code readability and maintainability
- Delivery notes and self-evaluation

## Course Rhythm

The course lasts 14 days. It is intentionally short, so assignments focus on producing practical artifacts rather than long theory essays.
